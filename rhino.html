Welcome to EnneadTab

EnneadTab is a powerful productivity suite designed for architects and designers, providing seamless integration with popular CAD and BIM software including Revit, Rhino, AutoCAD, and InDesign.

Software Integration

Revit Integration

Comprehensive tools for Revit automation, family management, and project optimization.

  • Family creation and management
  • Project automation tools
  • BIM optimization

Rhino Integration

Advanced modeling tools and utilities for Rhino 3D modeling and visualization.

  • Block management
  • Surface modeling tools
  • Rendering utilities

AutoCAD Integration

Streamlined CAD workflows with custom commands and automation features.

  • Custom LISP routines
  • Drawing automation
  • Layer management

InDesign Integration

Publishing and layout tools for professional documentation and presentation.

  • PDF processing
  • Layout automation
  • File management
